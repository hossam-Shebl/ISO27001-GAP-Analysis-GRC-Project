# ISO27001-GAP-Analysis-GRC-Project
ISO/IEC 27001:2022 GAP Analysis project including maturity assessment, risk insights, and remediation plan.
## 📌 Overview
This project presents a comprehensive GAP analysis conducted against ISO/IEC 27001:2022 for a simulated organization (Tech Solution Company).

The objective is to assess the current information security maturity level, identify gaps, and provide a structured remediation plan aligned with ISO 27001 requirements.
---
## 🎯 Key Objectives
- Evaluate ISMS maturity using a structured model (Level 0–5)
- Identify compliance gaps across ISO 27001 clauses
- Provide actionable remediation recommendations
- Support ISO 27001 certification readiness
---
## 🏢 Scope of Assessment
- IT Department
- Cybersecurity Department
- HR Department
- Finance Department
- Operations
---
## 📊 Methodology
The assessment was conducted using:
- Interviews
- Questionnaires
- Documentation review
- Observations
- Technical tools (e.g., Nessus)
---
## 📈 Maturity Model
Based on a 0–5 scale:
- 0 → Non-existent
- 1 → Initial
- 2 → Managed
- 3 → Defined
- 4 → Quantitatively Managed (Target)
- 5 → Optimized
---
## 🚨 Key Findings
- 2 Critical gaps identified
- 5 High-level gaps
- Major issues:
  - No internal audit program
  - No vendor management framework
